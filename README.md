# Mini-Shopping-Billing-System
MY MAJOR PROJECT ON C PROGRAMMING FOR 1ST SEMESTER.

Project Description: A simple C-based console application that manages products, calculates totals, and generates customer bills. It uses structures and file handling to store item details and produce a clear, accurate invoice for small shops.

How to Run:

Open src folder in terminal using 'cd src'.
Complie by 'gcc main.c shop.c -I ../include -o BILLING
Run by './BILLING'
Repository includes: -src/ consisting of .c files 'main.c' and 'shop.c' -include/ consisting of .h(header) files 'shop.h' -docs/ consisting of project report -README.md/ consisting of project description and instructions
